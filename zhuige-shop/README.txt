== Description ==

## 功能与特点

就是一个简单的商城小程序

微信：**jianbing2011**（加微信，进微信群交流）


== Installation ==

1. 把zhuige-shop文件夹上传到/wp-content/plugins/目录下

2. 在后台插件列表中启用"追格商城小程序"

3. 设置微信小程序的appid，及其他配置信息
